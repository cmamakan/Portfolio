Bellabeat

Phishing
This project aims to raise awareness about phishing by analyzing phishing trends. Using Python, a website listing phishing URLs was scraped with the help of BeautifulSoup and Requests. Various data were extracted from an XML file and converted into a CSV format.

This CSV was then loaded into Power BI to identify the most affected sectors and emerging trends. The goal is to educate users about phishing risks and promote better online security.
Sentiment Analysis
This project used natural language processing techniques to examine tweets about the highly debated pension reform in France, which led to significant public discussion and strikes.

Data was scraped from X and stored in MongoDB, with analysis performed using TextBlob, NLTK, and SpaCy, and processed with PySpark. A dashboard was then created using Flask to visualize the sentiment data and provide insights into public opinion.

Daily News
The goal of this project was to provide a daily news summary from the front pages of three prominent French newspapers. This involved scraping headlines, summaries, and content from multiple articles into a dataframe.

Langchain was utilized to provide an API for requesting daily news summaries, with this integration serving as an experiment to explore API technologies.
Salary prediction
This project aims to predict salaries for various data-related jobs based on different characteristics. The process involved scraping a file from a website to gather relevant salary data, followed by analyzing the data using Power BI.

Gradient Boosting was used to develop a predictive model. An API was created and tested using Postman, and then integrated into a Flask-based graphical interface, allowing users to select job characteristics and obtain salary predictions.
